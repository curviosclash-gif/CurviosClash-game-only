# Local Map Data

Dieser Ordner ist der lokale Ablagepfad fuer editor-generierte Map-Dateien.

Hinweise:
- Runtime-/Editor-Tools erwarten das Verzeichnis `data/maps`.
- QA-Temporaries wie `editor_qa-disk-map-*.editor.json` und `editor_qa-disk-map-*.runtime.json`
  sind in `.gitignore` ausgenommen und muessen nicht versioniert werden.
